## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd my-app
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

### Build for Production

```bash
npm run build
```

The built files will be in the `dist/` directory.

## ğŸ¨ Customization

### Colors

The color scheme is defined in `src/index.css` using CSS variables:

```css
:root {
  --background: 222 47% 6%;    /* Dark blue-gray */
  --primary: 217 91% 60%;       /* Bright blue */
  /* ... */
}
```

### Content

Update your personal information in `src/App.tsx`:
- Hero section (name, title)
- About section (bio)
- Skills section (technical skills)
- Experience section (work history)
- Projects section (project details)
- Education section (degrees, volunteering)
- Contact section (links)

### Images

Replace `public/profile.jpg` with your own professional photo.
